## 🧶 Página de Crochet  
Aplicación web tipo catálogo para mostrar y gestionar productos artesanales en crochet.  

### 🚀 Características
- Catálogo dinámico con búsqueda, filtros y ordenamiento  
- Vista detallada con carrusel de imágenes  
- Panel de administración protegido  
- Contacto directo para pedidos personalizados  

### 🛠️ Tecnologías
- **Frontend:** Next.js (App Router), React  
- **Estilos:** Tailwind CSS  
- **Backend / DB:** Supabase (PostgreSQL + Auth + Storage)  
- **UX:** Manejo de estado, modo oscuro, UI responsiva  

🔗 [Visitar el sitio](https://pagina-crochet.vercel.app/)

## 🎮 MathQuest - Juego de Matemáticas  
Plataforma educativa gamificada para aprender matemáticas de forma interactiva.  

### 🚀 Características
- Sistema de niveles, progreso y desbloqueos  
- Categorías (sumas, restas, multiplicaciones, etc.)  
- Logros (trofeos) y ejercicios interactivos  
- Sistema de autenticación y guardado de progreso  

### 🛠️ Tecnologías
- **Frontend:** Next.js, React  
- **Estilos:** Tailwind CSS  
- **Backend / DB:** Prisma ORM + PostgreSQL (Supabase)  
- **Autenticación:** Supabase Auth  
- **Arquitectura:** App Router + manejo de estado por usuario  

🔗 [Jugar ahora](https://mathquest-eight.vercel.app/)

## 🤖 StudAI - Generador de Videos con IA  
Aplicación web que convierte documentos PDF en videos explicativos con narración automática.  

### 🚀 Características
- Subida de PDF y generación automática de video  
- Pipeline con IA (extracción de texto → guion → voz → video)  
- Historial local de videos generados  
- Reproductor integrado y descarga de resultados  

### 🛠️ Tecnologías
- **Frontend:** Next.js (TypeScript), React  
- **Estilos / UI:** Tailwind CSS, Framer Motion  
- **Backend:** FastAPI (Python)  
- **IA:** Servicios de Azure (OpenAI, TTS)  
- **Procesamiento:** FFmpeg para renderizado de video  
- **Comunicación:** Axios + polling para procesos asíncronos  

🔗 [Probar aplicación](https://stud-ai-mocha.vercel.app/)
